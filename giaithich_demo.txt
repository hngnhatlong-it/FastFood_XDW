**Phần Demo Hoạt Động Hệ Thống Thanh Toán Điện Tử**

Hôm nay em sẽ demo một hệ thống API thanh toán đơn giản bằng QR code.
Hệ thống gồm 2 phần chính:

- Frontend: HTML + JavaScript để hiển thị website bán hàng.

- Backend: Node.js để tạo API thanh toán và sinh QR code VietQR.

Đây là giao diện website bán hàng.
Website có 3 sản phẩm:

- Laptop Gaming.

- Chuột Gaming.

- Bàn phím cơ.

Người dùng có thể thêm sản phẩm vào giỏ hàng và bấm Thêm.

Khi thêm sản phẩm, hệ thống sẽ cập nhật giỏ hàng và tổng tiền bằng JavaScript.

Khi người dùng bấm Thanh toán, hệ thống sẽ:

- Tạo mã đơn hàng ngẫu nhiên.

- Gửi dữ liệu lên API Node.js.

- API sẽ tạo QR thanh toán VietQR.

- QR được gửi lại cho frontend.

Đây là API backend được viết bằng Node.js và Express.

"app.post("/create-payment")"
-> API này nhận:

amount: số tiền<
orderId: mã đơn hàng

"https://img.vietqr.io/image/VCB-xxxx.png"
=> Link này sẽ tạo QR code chuyển khoản ngân hàng.

===>>>===>>>===>>> Nếu người dùng không có file nodejs thì hệ thống front-end khi đến bước thanh toán sẽ bị chờ vì không thể gửi API qua backend để tạo QR thanh toán. Nên vì thế phải cần file nodejs (backend) để tạo QR. <===<===<===

Sau khi API trả dữ liệu, frontend sẽ hiển thị QR code để người dùng quét thanh toán. Người dùng có thể dùng app ngân hàng để quét và chuyển tiền.

Sau khi khách hàng thanh toán, hệ thống sẽ:
-Xác nhận thanh toán thành công.

-Xóa giỏ hang.

-Reset đơn hàng.

Qua demo này, hệ thống đã mô phỏng được quy trình thanh toán cơ bản:

Người dùng chọn sản phẩm.

Tạo đơn hàng.

Backend tạo QR thanh toán.

Người dùng quét QR để trả tiền.

Đây là mô hình đơn giản của API thanh toán trong thương mại điện tử.
